## 📖 Contexto e Objetivos
Este projeto foi desenvolvido como parte de um desafio prático na **DIO (Digital Innovation One)**. O objetivo principal é explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa, utilizando o **NotebookLM** para organizar e sintetizar conhecimentos.

* **Tema Escolhido:** IA Generativa e Produtividade no Trabalho/Estudos.
* **Objetivo de Estudo:** Entender os pilares dos LLMs, técnicas de Grounding e como otimizar fluxos de trabalho com IA.

### 🩹 Cicatrizes (Troubleshooting)
* **Desafio:** Garantir que a IA não inventasse conceitos fora do guia fornecido.
* **Solução:** Ativei a funcionalidade de "Grounding" (ancoragem) do NotebookLM, selecionando especificamente a fonte do Google Doc para as respostas.
* **Lição Aprendida:** A curadoria de fontes é o passo mais importante para evitar alucinações da IA.

## 📝 Miniguia de Estudo (Entrega Final)

### 1. Resumo Estruturado
* **IA Generativa:** Diferente da preditiva, foca em criar novos conteúdos (texto, imagem, código).
* **Engenharia de Prompts:** É a habilidade técnica de desenhar comandos eficazes.
* **Produtividade:** O uso de IA permite focar em tarefas criativas e estratégicas enquanto a IA lida com a síntese de dados brutos.

### 2. Glossário de Conceitos
* **Tokens:** Unidades de processamento de texto.
* **Grounding:** Limitar o conhecimento da IA a um conjunto específico de dados fiáveis.
* **Chain-of-Thought:** Técnica de prompt que pede à IA para detalhar o raciocínio passo a passo.

### 3. Toolkit de Prompts Reutilizáveis
* `"Atue como um tutor e explique o conceito [X] de forma simples."`
* `"Extraia as 5 palavras-chave mais importantes do documento fornecido."`
* `"Crie um roteiro de revisão semanal baseado nos tópicos deste caderno."`
